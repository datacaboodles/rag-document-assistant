
Coffee Machine Cleaning Standard Operating Procedure

Purpose:
To ensure hygiene, beverage quality, and machine longevity.

Daily Cleaning Steps:
1. Turn off the machine and unplug it.
2. Remove drip tray and waste container.
3. Wash removable parts with warm water and mild soap.
4. Wipe external surfaces with a damp cloth.
5. Reassemble all parts and restart.

Weekly Deep Cleaning:
1. Run the internal cleaning cycle using the machine menu.
2. Use approved cleaning tablets — do not substitute.
3. Rinse all milk lines with clean water.
4. Inspect nozzles for blockage and clear if needed.

Descaling:
- Perform descaling every 30 days.
- Use only the approved descaling solution (SKU: DESC-001).
- Never interrupt the descaling cycle once started.

Troubleshooting:
- Bitter coffee: Clean the brew group immediately.
- Slow dispensing: Check nozzle for blockage.
- Error code E12: Water tank is empty — refill before use.
- Error code E05: Drip tray full — empty and reinsert.
